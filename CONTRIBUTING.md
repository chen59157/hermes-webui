# 贡献指南

感谢你对 Hermes WebUI Desktop Client 的关注！

## 如何贡献

### 报告 Bug

1. 在 GitHub Issues 中搜索是否已有相同问题
2. 提供详细的复现步骤
3. 附上 `hermes.log` 日志文件（脱敏后）
4. 注明操作系统版本和 Python 版本

### 提交代码

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/your-feature`
3. 提交变更：`git commit -m "feat: 描述你的变更"`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

### 提交规范

使用 [Conventional Commits](https://www.conventionalcommits.org/) 格式：

```
feat: 新功能
fix: 修复 Bug
docs: 文档更新
style: 代码格式调整
refactor: 重构
test: 测试相关
chore: 构建/工具链变更
```

### 开发环境

详见 [docs/development.md](docs/development.md)。

## 行为准则

- 尊重所有贡献者
- 建设性的代码审查
- 保持讨论聚焦于技术问题
