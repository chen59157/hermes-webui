Hermes WebUI - Windows 桌面客户端
================================
版本: 3.2.0
更新日期: 2026年6月5日

一、快速启动
-----------
  双击桌面「Hermes WebUI」快捷方式即可运行。
  已内置 Python 运行时，无需单独安装 Python。
  启动时无黑窗闪烁，直接显示 Loading 过渡页。

二、系统要求
-----------
  - Windows 10/11 64位
  - 可用磁盘空间: 500 MB+
  - 网络连接（API 调用需要）

三、首次使用
-----------
  1. 双击桌面「Hermes WebUI」快捷方式
  2. 等待 Loading 页面显示启动进度（约 3-5 秒）
  3. 主窗口自动加载 http://127.0.0.1:8787
  4. 在设置页面配置 API Key（如已有配置则自动加载）

四、v3.2.0 新功能
-----------------
  【性能优化】
  - Python 路径缓存：启动速度提升 30-50%
  - 日志系统升级：RotatingFileHandler 自动轮转（10MB x 3）
  - 启动流程优化：减少固定等待时间约 2.7 秒

  【Hermes Agent 升级至 v0.16.0】
  - 模型选择器支持模糊搜索
  - /undo [N] 回退最近 N 轮对话
  - 默认技能集精简，减少上下文占用
  - 安全修复：2 个 P0 + 62 个 P1 + 16 个安全修复
  - 新增 FastAPI/Starlette 依赖（Web Dashboard 支持）

  【语言设置优化】
  - 支持 10 种语言：中文、English、日本語、한국어、Русский 等
  - 设置面板语言选择即时生效
  - 语言偏好自动保存

  【版本信息】
  - 设置页面显示 WebUI 和 Agent 版本号
  - 支持检查更新

  【UI/UX 升级】
  - 全新设计系统：统一的 CSS 变量（颜色/字体/间距/圆角/阴影）
  - UI 组件库：Toast 通知、Dialog 弹窗、Loading 组件
  - 全新 Loading 过渡页：双环旋转动画 + 进度条

  【右侧思考面板】
  - 实时展示 AI 的思考过程（需模型支持 reasoning）
  - 支持折叠/展开，点击标题栏切换
  - 面板折叠时右边缘显示展开按钮（箭头）

  【平台配置功能】
  - 助理设置页面支持配置 7 个平台（飞书/微信/QQ/钉钉等）
  - 配置保存到 ~/.hermes/platforms.json

  【原生 Toast 通知】
  - 权限请求和需要回复时，右下角弹出 Windows 原生通知
  - 即使软件最小化到托盘也能看到

  【复制功能增强】
  - 消息复制按钮始终可见（半透明显示）
  - 消息文本可自由选中复制
  - WebView2 环境下文本选择兼容性优化

五、自定义缓存位置
-----------------
  缓存数据（日志、会话、配置）默认保存在用户目录的 .hermes/ 文件夹。

  如需自定义位置：
  1. 打开 Hermes WebUI
  2. 进入设置页面
  3. 修改「缓存目录」路径
  4. 保存后生效（现有数据会自动迁移到新位置）

六、开机自启动
-------------
  方式一：在应用内开启
    - 打开 Hermes WebUI → 设置 → 开启「开机自启动」

  方式二：手动设置
    1. 按 Win+R 打开运行
    2. 输入 shell:startup 回车
    3. 将「Hermes WebUI」快捷方式放入该文件夹

七、系统托盘
-----------
  应用运行后会在系统托盘（右下角）显示图标。
  - 左键点击：显示/隐藏主窗口
  - 右键点击：打开菜单（重启服务、设置、退出）
  - 关闭窗口不会退出程序，会最小化到托盘继续运行

八、常用端口信息
---------------
  - 默认端口: 8787
  - 访问地址: http://127.0.0.1:8787
  - 健康检查: http://127.0.0.1:8787/health

九、故障排除
-----------
  Q: 页面显示"You are offline"？
  A: 后端服务未启动。查看 logs/hermes.log 日志文件，或重新启动。

  Q: 端口被占用？
  A: 应用会自动检测并尝试其他端口。也可手动修改设置中的端口。

  Q: 如何完全卸载？
  A: 通过控制面板「程序和功能」卸载，或运行开始菜单中的卸载程序。

十、目录结构
-----------
  C:\Program Files\Hermes WebUI\
  ├── hermes_desktop.py         ← 桌面客户端主程序
  ├── hermes-webui-cn/          ← WebUI 前端 + Flask 后端
  ├── hermes-agent/             ← Agent 核心引擎（含 .venv 虚拟环境）
  ├── hermes-warden/            ← 任务守护进程
  ├── python-runtime/           ← 内置 Python 3.12 运行时
  ├── tk_runtime/               ← 内置 Tk 运行时
  ├── logs/                     ← 运行日志（hermes.log）
  ├── hermes-icon.ico           ← 应用图标
  ├── design-system.css         ← 设计系统变量
  ├── ui-components.css         ← UI 组件样式
  ├── ui-components.js          ← UI 组件逻辑
  ├── loading.html              ← Loading 过渡页
  └── 使用说明.txt              ← 本文件

  用户数据目录:
  ~/.hermes/
  ├── webui/settings.json       ← 用户设置
  ├── webui/platforms.json      ← 平台配置
  ├── webui/sessions/           ← 会话数据
  ├── logs/server.log           ← 后端服务日志
  └── config.yaml               ← Agent 配置

十一、技术支持
-------------
  日志文件: logs/hermes.log
  服务日志: ~/.hermes/logs/server.log
